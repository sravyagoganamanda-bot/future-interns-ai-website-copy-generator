# AI Website Copy Generator for Local Businesses

## Project Overview
This project was created as part of the Future Interns Prompt Engineering Internship Task 1.

The goal of this project is to design structured AI prompts that generate high-quality website copy and website designs for local businesses.

## Business Chosen
Velvet Brew Cafe — Hyderabad

## Tools Used
- ChatGPT
- Lovable AI
- GitHub

## Features
- AI-generated cafe website
- Homepage copy generation
- Services page content
- CTA generation
- Modern responsive design
- Prompt engineering workflow

## Prompt Logic
A detailed master prompt was designed to generate:
- website structure
- branding
- visual layout
- CTA sections
- business-focused website copy

## Output
The project successfully generated a realistic and visually appealing cafe website using AI-powered prompt engineering.
